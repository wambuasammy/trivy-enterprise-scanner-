# Changelog

## v2.0.0 - Initial Release

- Automated Docker image scanning with Trivy
- Automatic import from /media/sf_Documents/
- Interactive image selection
- JSON, CSV, Summary CSV, HTML, and ZIP reports
- Severity support: CRITICAL, HIGH, MEDIUM, LOW
- Bulk scanning (--all)
- Cleanup mode (--cleanup)
- Logging to scan.log
- CI/CD-friendly exit codes
